# Product

## Register

product

## Users

Precompile Studio is for developers and onchain builders experimenting with Ritual testnet precompiles. They are likely switching between docs, explorer, wallet, and code while trying to prepare a correct call before spending gas.

## Product Purpose

The product helps users compose, inspect, and prepare Ritual precompile calls with visible readiness checks. Success means a builder can understand whether their wallet, chain, balance, ABI input, and guardrails are ready before moving to a signed transaction or consumer flow.

## Brand Personality

Technical, calm, precise.

The interface should feel close to Ritual Explorer: dark, dense, quiet, and trustworthy. It should not feel like a generic dashboard, marketing site, or decorative SaaS landing page.

## Anti-references

- Generic Web3 dashboards with oversized metric cards and chart sprawl.
- AI-looking SaaS pages with decorative gradients, grid overlays, floating cards, and loud hero sections.
- Interfaces that make guardrails feel like errors before the user has acted.
- Designs that mimic Ritual Explorer visually but ignore the workflow of composing a precompile call.

## Design Principles

- Composer first: the main task is preparing a correct precompile request, so the composer should dominate the page.
- Readiness supports action: status checks should be compact and useful, not the main visual event.
- Calm technical density: prefer restrained typography, predictable controls, and compact panels over spectacle.
- Evidence over decoration: every visible element should help the user understand inputs, readiness, ABI shape, or next steps.
- Match Ritual’s ecosystem: borrow density, typography rhythm, dark surfaces, and quiet state treatment from Ritual Explorer without turning this into another explorer.

## Accessibility & Inclusion

Use WCAG AA as the baseline. Preserve visible focus states, avoid color-only status communication, support reduced motion, and keep technical values readable at desktop and mobile widths.
